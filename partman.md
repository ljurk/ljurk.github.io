# partman

## Part Managment tool

* built with docker
    * postgres
    * python
    * php
* cli with python


just build
    
    docker-compose up --build
    
and run
    
    docker-compose up
    
you can see your api under
    
    http://localhost:5001
    
and your website under
    
    http://localhost:5000
